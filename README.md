# MONSAM Mobile

App React Native (Expo) para gestionar pedidos de aretes MONSAM desde el celular.

## Requisitos
- Node.js 18+
- Expo Go instalado en el celular (Play Store)
- monsam-app corriendo en la PC (puerto 4000)
- Celular y PC en la misma red WiFi

## Instalación
```bash
npm install
npx expo start
```

Escanea el QR con Expo Go.

## Configuración
1. Abre la app → pestaña **Config**
2. Ingresa la URL de la PC (ej: `http://192.168.1.x:4000`)
3. Presiona **Probar conexión**
